# ADR 0004: SignalR per notifiche realtime

## Status
Accepted

## Context
Gli operatori devono ricevere aggiornamenti in tempo reale senza polling.

## Decision
Usare SignalR come canale realtime esposto da web-backend-core, integrato con notifier.
